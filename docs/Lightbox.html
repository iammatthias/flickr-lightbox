Class: Lightbox

Lightbox

new Lightbox(photos, parentElem)

Lightbox represents the lightbox component. It displays a picture, has left and rightElem arrows to navigate between pictures and shows a preloader while an image is loading.
Parameters:
Name Type Description

onClickHandler(e)

Lightbox.onClickHandler handles what happens when the Lightbox is clicked. If the arrows are clicked, the next or previous picture is displayed. If the dark area (outside the Lightbox content) is clicked, the Lightbox is hidden.
